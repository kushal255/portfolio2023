"Exploration of FAB-LAB Course Module 1 : Uderstandind Web Developement"



This report highlights my experience during the seven-day Fab-Lab module one course at the college campus, focusing on "Understanding Web Development."
Over the span of this program, I delved into the fundamentals of web development, including essential languages like HTML, CSS, and JavaScript.
The course thoroughly covered both front-end and back-end development aspects, providing a well-rounded understanding.


Throughout the module, I engaged in various hands-on activities and projects that solidified my learning.
One of the remarkable outcomes of this course was the creation of a functional website, which serves as a testament to the skills acquired.


The journey into web development was both enlightening and rewarding.
Learning about HTML empowered me to structure the content of web pages effectively, while CSS allowed me to style them in visually appealing ways. JavaScript, on the other hand, enabled me to add dynamic and interactive elements to websites.


In conclusion, the module was a comprehensive introduction to web development.
The knowledge gained about HTML, CSS, and JavaScript, along with the practical experience of creating a website, has equipped me with valuable skills for crafting engaging and functional web pages.
This foundation sets the stage for further exploration and growth in the realm of web development.
